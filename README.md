# ChemQuest 10 ⚗️

An interactive, game-based playground for the first 4 chapters of NCERT Class 10 Science (Chemistry). Built for visual learners — no walls of text, just play.

## Modules

1. **The Recipe Balancer** (Chemical Reactions & Equations) — balance real NCERT equations on a tilting scale with visual atom weights.
2. **The Litmus Color Lab** (Acids, Bases & Salts) — drag substances onto a litmus strip, watch it change colour, and read the friendly pH gauge.
3. **The Material Sorting Game** (Metals & Non-Metals) — sort items and properties from a moving conveyor belt into Team Metals or Team Non-Metals.
4. **Carbon's Hand-Holding Club** (Carbon & its Compounds) — fill Carbon's 4 empty hands with hydrogens to build methane, and explore single/double/triple bonds.

## Run it

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Stack

Next.js 14 (App Router) · Tailwind CSS · Framer Motion
